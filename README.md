# Mediator Pattern — TypeScript

## Опис

Цей проєкт демонструє реалізацію поведінкового патерну проєктування Mediator мовою програмування TypeScript.

Патерн Mediator використовується для зменшення прямої залежності між об'єктами. Замість того, щоб компоненти взаємодіяли один з одним напряму, вони спілкуються через посередника — mediator.
